What is the Interplanetary Wiki?

Interplanetary Wiki allows you to create a set of Wiki pages that link to each other.

Interplanetary Wiki pages are stored on IPFS, the Interplanetary Filesystem.

It is built using just IPFS and Javascript - the simplest possible thing that could work. "Wiki" means "quick" in Hawaiian and the Wiki goal is that it is quick to edit the web.

Start your Wiki by following these steps:

  1. Enter some text in the sandbox below.
  2. Link to a new page that you wish to create by typing its names using 2 or more words joined together and with the first letter capitalised, like HomePage or ToDoList. This will automatically create a link.
  3. Click on the link, and a page editor will be presented in a brand new Context. The Context is the "umbrella" for all of the Wiki pages created. Any Wiki page can link to any other in the same context by simply adding text containing the name of the target page in special WikiName format.
  4. After you have finished editing the page, click on the Save button and the page will be saved to the IPFS network.
